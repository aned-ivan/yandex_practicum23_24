# Аналитика данных сервиса аренды самокатов

Проект посвящен аналитике данны популярного сервиса аренды самокатов GoFast.
Чтобы совершать поездки по городу, пользователи сервиса GoFast пользуются мобильным приложением.
Исследуются данные о некоторых пользователях из нескольких городов, а также об их поездках. 

#### Цель проекта:
Проанализировать данные и проверить некоторые гипотезы, которые могут помочь бизнесу вырасти.
